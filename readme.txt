# Garen King

Garen King é uma aplicação web dedicada ao campeão Garen, de League of Legends. O projeto reúne informações úteis para jogadores que desejam consultar build, runas, matchups, evolução do campeão, itens, skins, história e referências de alto elo.

## Funcionalidades

* Guia de build para Garen
* Runas recomendadas
* Ordem de habilidades
* Feitiços de invocador
* Principais matchups do topo
* Classificação de confrontos favoráveis, neutros e difíceis
* Linha do tempo com mudanças importantes do campeão
* Evolução dos principais itens por era
* História e lore do personagem
* Área de skins
* Seção de notícias
* Interface visual temática de Demacia
* Layout organizado e responsivo

## Objetivo

O objetivo do projeto é oferecer uma página completa e visualmente atrativa para jogadores de Garen, centralizando informações importantes sobre gameplay, história, builds e evolução do campeão dentro do League of Legends.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Netlify
