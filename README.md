# Garen King

> Guia interativo completo do campeao Garen para League of Legends.

## Sobre o projeto

Aplicacao web tematica dedicada ao campeao Garen de League of Legends. Centraliza build recomendada, runas, ordem de habilidades, feiticos de invocador, matchups do topo, linha do tempo de mudancas do campeao, evolucao de itens, historia e lore, area de skins e noticias. Visual inspirado em Demacia com layout responsivo.

## Funcionalidades principais

- Guia de build, runas e ordem de habilidades
- Classificacao de matchups favoraveis, neutros e dificeis
- Linha do tempo com mudancas importantes do campeao
- Evolucao dos principais itens por era
- Historia, lore e galeria de skins
- Secao de noticias e interface tematica Demacia

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Netlify

## Como executar

Abra index.html no navegador.
